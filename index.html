About Me

I'm Samuel Chibinji Mwanza, a passionate Software Engineer/Data Scientist and final-year Computer Science student at the University of Zambia. My journey into software engineering and data science began with a curiosity about how technology can solve real-world problems in my home country of Zambia.

My background combines formal education in computer science with self-directed learning in modern technologies. I'm particularly interested in educational technology and healthcare systems, believing these sectors have immense potential for digital transformation in Zambia. I've developed applications that address local challenges, from pharmacy management to student performance tracking.

What excites me most about software development is the ability to create solutions that reach people where they are. Whether it's a teacher in a rural school checking student progress or a pharmacist managing inventory, well-crafted software applications can make a real difference. My goal is to continue building tools that are not only technically sound but also accessible and meaningful to Zambian users.

I believe my unique perspective as a Zambian student developer allows me to understand both the technical requirements and the local context needed to create truly effective solutions. I'm eager to collaborate on projects that combine technology with social impact, particularly in education and healthcare.

My Projects

Pharmacy Point of Sale System

A comprehensive POS system designed for Zambian pharmacies. Features include inventory tracking, expiry date management, sales reporting, and low-stock alerts. Built with React for the frontend and Node.js with MongoDB for the backend. Includes role-based access for pharmacists and cashiers. This system helps pharmacies reduce medication waste through expiry tracking and optimize inventory levels.

Live Demo GitHub

ZamPrep: Parent Classroom Performance Tracker

A comprehensive web and mobile application that allows parents to monitor their children's academic progress in real-time. Teachers can update grades, attendance records, and behavioral notes, while parents receive weekly summaries and instant alerts about their child's performance. Features interactive data visualizations showing performance trends across subjects, comparative analytics with class averages, and personalized recommendations for improvement. Built with React Native for mobile, Django for backend, and PostgreSQL for database. Includes offline sync capability for schools with limited internet connectivity.

Live Demo GitHub

University Student Dropout Risk Predictor

A machine learning-powered application that identifies students at risk of dropping out from Zambian universities. The system analyzes academic records, attendance patterns, socioeconomic factors, and engagement metrics to generate early warning risk scores. Provides actionable insights and intervention recommendations for university counselors and academic advisors. Includes a dashboard for visualizing risk factors across student populations and tracking intervention effectiveness. Built with Python, scikit-learn for machine learning models, Flask for the API, and React for the frontend dashboard.
